What are the best practices associated with using classes vs. ids?

Using ids and classes is important when writing html code because it helps to keep a uniform look to your website.

1 March 2025

What is an id?
Using an id allows you to identify a single element of your HTML code and apply a specific style to that element. Each of the elements in your HTML page may only have one id connected to them and each time you add an id to a HTML element it must be different to the other ids you have added on that page already. No ids on that page can be the same.

So then, what's a class?
A class is similar to an id although you can use the same class for multiple elements on the same HTML page or even the entire website. Elements can have multiple classes attached to them. A class is an easy way to identify multiple elements and apply the same style to each of those elements.

To identify classes in your CSS file, you can use a . (full stop) followed by the name of the the class:

If you get stuck thinking about classes vs ids, remember this:
Classes can be used multiple times - just like a class has multiple students.
Ids are unique - just like a drivers license which should only belong to one person.
